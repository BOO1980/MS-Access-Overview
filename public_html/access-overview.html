Access Overview

  • Microsoft Access is a Database Management System (DBMS)
  • Access combines the relational Microsoft Jet Database Engine with A Graphical user interface and software development tools
  • Access is part of the Microsoft office suite of applications
  • An Access database is an automated version of filing and retrieval functions of a paper filing system
  • Access tables store a variety of different kinds of data, from simple lines of text to complex data
  • Storing data in a precise format enables a database management system like Access to turn data into useful information
  • Tables serve as the primary data repository in an Access database
  • Queries, forms and reports provide access to the data, enabling the user to add or extract data, and presenting the data in useful ways
  • Most developers add macros or Visual Basic for Applications (VBA) code to forma and reports to make their Access applications easier to use
  • A relational database management systems (RDBMS), such as Access, stores data in related tables
  • Queries allow the user to ask complex questions, with the answers displayed as on screen forms and printed reports
  • One of the fundamental different between a relational database and a manual filing system, a relational database's data for a single person or item may be stored in separate table
  • In Access, a database is the overall container for the data and associated objects
  • Access allows you to create Custom web app, Web apps are used and shared in browsers
